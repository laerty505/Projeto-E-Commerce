Descrição do sistema:

A CoreShop é um projeto de loja de comercialização de placas de vídeo (GPUs), desenvolvida como projeto acadêmico da disciplina de Programação Web I do curso de Análise e Desenvolvimento de Sistemas do IFPE.

O sistema conta com páginas de cadastro de usuário, login e recuperação de senha (Sem validação), garantindo um fluxo básico de acesso à plataforma. O usuário pode navegar pelo catálogo de GPUs disponíveis e adicionar produtos ao carrinho de compras.

Todo o projeto foi desenvolvido utilizando HTML como base, com pequenos ajustes feitos em CSS (No cabeçalho) para melhoria visual da interface.

Dificuldades encontradas:

A principal dificuldade encontrada durante o desenvolvimento foi a estilização do cabeçalho, especialmente o posicionamento dos botões de navegação (Carrinho, Sobre e Sair) no lado direito da tela, o que exigiu o uso de CSS para alinhar os elementos corretamente.
